# komiti
M Plus 1p と Titillium Web を合成した日本語 TrueType フォント

### komitiのなかみ
- **半角数字** + 日本語 -> M PLUS 1p
- 半角英字 + 半角記号 -> Titillium Web

### Titillium Webの改変点
`//` スラッシュ同士のカーニングを変更
